# Gemini Agent Skills

This directory contains a collection of specialized agent skills for software engineering and productivity. These skills are designed to be composable and easy to adapt for various AI agents.

## Contents

- **engineering/**: Skills for debugging, architecture, TDD, and issue management.
- **productivity/**: Workflow tools like grilling sessions and token-efficient communication.
- **misc/**: Helpful utilities for git, exercises, and project setup.
- **personal/**: Personal workflow and note-taking skills.

## Usage

To use these skills with your Gemini CLI agent, you can refer to the `SKILL.md` files within each subdirectory. Each skill provides structured instructions and best practices for specific tasks.

*Note: These skills were consolidated from the local skills repository for easy version control and cross-system reuse.*
